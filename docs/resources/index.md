## Jak dodawać zasoby?

Stwórz wpis zawierający:

```markdown
### Nazwa Zasobu

- **Typ:** Artykuł / Kurs / Narzędzie / etc.
- **Przedmiot:** Do którego przedmiotu się odnosi
- **Link:** [Tutaj](https://example.com)
- **Opis:** Krótkie streszczenie czym jest ten zasób
- **Dlaczego jest przydatny:** Jak to Ci pomaga w nauce?

```
