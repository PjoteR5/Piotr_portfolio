## Porada: Szybkie edytowanie

Aby szybko edytować notatki, możesz:

1. Otworzyć folder `docs/notes` w edytorze (np. VS Code)
2. Edytować pliki `.md` bezpośrednio
3. Uruchomić `mkdocs serve` i śledzić zmiany w przeglądarce
4. Commitować zmiany do Git
